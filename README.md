# Coffee Machine ☕

A simple Python coffee machine simulation project.

## Features
- Order espresso, latte, or cappuccino
- Checks ingredient availability
- Processes coin payments
- Returns change
- Tracks profit
- Generates resource report

## Concepts Used
- Functions
- Loops
- Dictionaries
- Conditionals
- State management

## Run the Program

```bash
python coffee_machine.py
```

## Commands
- espresso
- latte
- cappuccino
- report → shows resources and profit
- off → turns off machine
